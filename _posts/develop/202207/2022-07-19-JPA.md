# JPA란?

- JPA는 Java Persistence API. 자바의 영속성(지속성) API... 조금만 더 알아보자.
- JAVA APP <-> JDBC 사이에서 동작한다.
- JPA를 통해서 개발자가 데이터를 다루기떄문에 **JDBC를 직접 사용하지 않아도 된다.**
- 위의 특징으로 인해서 SQL 중심의 개발에서 **객체 중심의 개발**로 바뀌게 된다.
- 현재 자바 진영의 **ORM 기술 표준**
